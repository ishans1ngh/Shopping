# Cart-Based Price Comparison

## Overview
A simple static web application built with plain HTML, CSS, and JavaScript.

## Project Structure
```
public/
  index.html    - Main HTML file
  styles.css    - CSS styles
  script.js     - JavaScript code
server.cjs      - Simple Node.js static file server
```

## Running the Application
The application runs on port 5000 using a simple Node.js HTTP server that serves static files from the `public/` directory.

Command: `node server.cjs`

## Deployment
Configured for static deployment using the `public/` directory.
